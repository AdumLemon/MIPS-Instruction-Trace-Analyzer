CS370 Project 2 – Cache Simulator
Author: James Volkmann
Date: 5/10/2026

PROGRAM DESCRIPTION
This project reads 2 input files, "accesses.txt", and "parameters.txt", and simulates cache behavior and outputs statistics to "statistics.txt". It can handle 1, 2, and 4-way cache's, with Write Allocate, Write No Allocate, Write Through, and Write Back.

HOW TO COMPILE THE PROGRAM
Using GCC on the uwlax.compute.cs.server or another Linux or bash server, make sure there are 3 files named "accesses", "parameters", and "statistics", use "gcc main.c" in the directory with all of these files.

HOW TO RUN THE PROGRAM
Run the program using "./<name of compiled file>", this will read the input files and generate the statistics file.
